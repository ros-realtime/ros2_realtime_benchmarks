How to run the demo

Run the pendulum demo wuth the following settings: - update period (-u): 1 millisecond - maximum heap prefault size for dynamic memory (-d): 200 MB - real-time loop iterations (-i): 60000 - thread priority (-t): 90 - sched policy (-s): SCHED_FIFO

pendulum_demo -d 200mb -t 90 -s fifo -u 1ms -i 60000 -f pendulum_demo_results

See all the options available in the rttest repository.

How to generate the plots

rttest_plot pendulum_demo_results
